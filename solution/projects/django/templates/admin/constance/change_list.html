About Django Constance

Django Constance integration in django-cfg allows you to define runtime configuration fields directly in your config.py using ConstanceConfig and ConstanceField. Fields are automatically registered at Django startup and grouped into collapsible fieldsets. All settings are stored in the database and can be modified at runtime without restarting the application.

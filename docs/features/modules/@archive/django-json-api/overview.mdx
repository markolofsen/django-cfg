# JSON:API

<Callout type="info">
There is no `django_json_api` module in Django-CFG. This page describes how to achieve JSON:API compliance using the `djangorestframework-jsonapi` third-party package alongside Django-CFG's DRF integration.
</Callout>

## Current Status

Django-CFG does not include a built-in JSON:API module. The codebase has no `django_json_api` package under `modules/`. If you need JSON:API compliance in a Django-CFG project, use a third-party library alongside Django-CFG's DRF integration.

## JSON:API with `djangorestframework-jsonapi`

The most widely used library for JSON:API compliance in Django is [`djangorestframework-jsonapi`](https://django-rest-framework-json-api.readthedocs.io/). It works alongside Django REST Framework, which is already supported by Django-CFG.

### Installation

```bash
pip install djangorestframework-jsonapi
```

### Configuration

```python
REST_FRAMEWORK = {
    'DEFAULT_RENDERER_CLASSES': [
        'rest_framework_json_api.renderers.JSONRenderer',
        'rest_framework.renderers.BrowsableAPIRenderer',
    ],
    'DEFAULT_PARSER_CLASSES': [
        'rest_framework_json_api.parsers.JSONParser',
        'rest_framework.parsers.FormParser',
        'rest_framework.parsers.MultiPartParser',
    ],
    'DEFAULT_METADATA_CLASS': 'rest_framework_json_api.metadata.JSONAPIMetadata',
    'DEFAULT_FILTER_BACKENDS': [
        'rest_framework_json_api.filters.OrderingFilter',
        'rest_framework_json_api.django_filters.DjangoFilterBackend',
        'rest_framework.filters.SearchFilter',
    ],
    'DEFAULT_PAGINATION_CLASS': (
        'rest_framework_json_api.pagination.JsonApiPageNumberPagination'
    ),
    'EXCEPTION_HANDLER': 'rest_framework_json_api.exceptions.exception_handler',
}
```

### Serializer

```python
from rest_framework_json_api import serializers
from .models import Article

class ArticleSerializer(serializers.ModelSerializer):
    class Meta:
        model = Article
        fields = ('id', 'title', 'body', 'author', 'created_at')
```

### Response Format

JSON:API responses follow the `{data: {id, type, attributes, relationships}}` envelope:

```json
{
  "data": {
    "type": "articles",
    "id": "1",
    "attributes": {
      "title": "Introduction to JSON:API",
      "body": "..."
    },
    "relationships": {
      "author": {
        "data": {"type": "people", "id": "42"}
      }
    }
  }
}
```

## Django-CFG DRF Integration

Django-CFG provides DRF configuration through `DjangoConfig`. You can add `REST_FRAMEWORK` settings there while using `djangorestframework-jsonapi` for the JSON:API layer:

```python
from django_cfg import DjangoConfig

class MyConfig(DjangoConfig):
    # DRF settings are passed through to Django settings
    # Configure REST_FRAMEWORK in your settings module or DjangoConfig
    pass
```

See the [DRF Guide](../../../drf-guide) for details on how Django-CFG integrates with Django REST Framework.
